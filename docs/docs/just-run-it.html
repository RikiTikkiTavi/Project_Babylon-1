And how can i run this stuff?

by GreenElephant team

Okay. Let's begin...

First you need to clone or download our repo. You can do this by typing folowing to Unix-terminal (for Windows will be later): 

git clone https://github.com/RikiTikkiTavi/Project_Babylon-1.git

Then you need to change your directory to Project

cd Project_Babylon-1

And run awesome game! 

java -jar Babylon.jar

Be happy! ..... and don't forget to press space
